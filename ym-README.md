# Tasks Manager

### OVERVIEW

Task Manager will be used to create new tasks and keep a record of all tasks that have been loaded.

### FEATURES

- Home (main page)
- NavBar (for navigations)
- Task Log (list of tasks)
- Task Form (to submit new tasks)
- Submit button

### MVP

- Home Page
- Task Log Page
- Task Form Page
- Navigations
- Submit button

### STRETCH GOALS

- Task Details
- Delete button
- Update button
